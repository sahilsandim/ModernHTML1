HTML Entity some information

HTML entities are text strings that start with an ampersand (&) and end with a semicolon (;). They are used to display reserved characters and invisible characters. Reserved characters are characters that would otherwise be interpreted as HTML code. Invisible characters include non-breaking spaces.

HTML Entity Example

The copyright sign: ©

The less-than sign: <

The greater-than sign: >

The ampersand (and) sign: &

The registered trade mark sign: ®
